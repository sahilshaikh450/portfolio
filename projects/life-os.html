LIFE OS

LIFE OS is a futuristic full stack productivity and personal management platform built to help users organize their daily life efficiently. The platform combines habit tracking, task management, expense analytics, secure authentication, and productivity systems into one modern ecosystem with a premium user experience.

Features

  • Secure JWT authentication system
  • Habit tracking and streak management
  • Task and productivity management
  • Expense tracking and budgeting
  • Interactive analytics dashboard
  • Responsive modern UI design
  • Protected routes and authorization
  • REST API architecture
  • Persistent database integration
  • Modern full stack architecture

Tech Stack

React JavaScript Zustand Axios Java Spring Boot Spring Security JWT H2 Database JPA
