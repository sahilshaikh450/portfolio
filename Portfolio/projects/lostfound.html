AVIRO

AVIRO – Affordable, Verified & Intelligent Ride Organizer is a smart college ride-sharing platform designed for students. The platform helps students find affordable, safe, and convenient rides by connecting students traveling on similar routes and timings. It reduces travel expenses, fuel wastage, traffic congestion, and promotes eco-friendly transportation through verified student-based ride sharing.

Features

  • Student verification system
  • Ride booking and ride sharing
  • Route-based ride matching
  • Student-to-student connectivity
  • Affordable and secure transportation
  • Responsive user interface
  • Fast and interactive experience
  • JWT authentication and security

Tech Stack

HTML CSS JavaScript Java Spring Boot JWT Security MySQL
